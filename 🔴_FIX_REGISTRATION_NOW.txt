═════════════════════════

STEP 1: Create Account
──────────────────────
1. Open: https://www.mongodb.com/cloud/atlas
2. Click: "Try Free" button
3. Sign up with:
   • Google (fastest - 1 click)
   • GitHub (fast - 1 click)
   • OR Email (fill form)

STEP 2: Create Cluster
──────────────────────
After login, you'll see "Create a deployment"

1. Click: "Create" or "Build a Database"
2. Choose: "M0 FREE" tier
   ┌─────────────────────────────────────────────────────────────────────┐
   │ M0                                                                   │
   │ FREE                                                                 │
   │ Shared RAM, 512 MB Storage                                          │
   │ [SELECT THIS ONE] ← Click here                                      │
   └─────────────────────────────────────────────────────────────────────┘
3. Provider: AWS (recommended) or Google Cloud or Azure
4. Region: Choose closest to you
5. Cluster Name: Leave default or name it "FitAI"
6. Click: "Create Cluster" (takes 1-3 minutes to provision)

STEP 3: Create Database User
─────────────────────────────
While cluster is creating:

1. Click: "Database Access" (left sidebar)
2. Click: "Add New Database User"
3. Authentication Method: "Password"
4. Fill in:
   Username: fitai
   Password: fitai123
   (Or use your own - remember it!)
5. Database User Privileges: "Atlas admin"
6. Click: "Add User"

STEP 4: Allow Network Access
─────────────────────────────
1. Click: "Network Access" (left sidebar)
2. Click: "Add IP Address"
3. Click: "Allow Access from Anywhere"
4. IP Address: 0.0.0.0/0 (should auto-fill)
5. Comment: "Allow all IPs"
6. Click: "Confirm"

STEP 5: Get Connection String
──────────────────────────────
1. Click: "Database" (left sidebar)
2. Wait for cluster to finish creating (green status)
3. Click: "Connect" button on your cluster
4. Choose: "Connect your application"
5. Driver: Node.js
6. Version: 4.1 or later
7. Copy the connection string:

   It looks like:
   mongodb+srv://fitai:<password>@cluster0.xxxxx.mongodb.net/?retryWrites=true&w=majority

8. IMPORTANT: Replace <password> with your actual password (fitai123)

   Example:
   mongodb+srv://fitai:fitai123@cluster0.abc123.mongodb.net/?retryWrites=true&w=majority

STEP 6: Update Your Configuration
──────────────────────────────────
1. Open file: server/.env
2. Find this line:
   MONGODB_URI=mongodb://localhost:27017/fitness-ai

3. Replace with your connection string:
   MONGODB_URI=mongodb+srv://fitai:fitai123@cluster0.xxxxx.mongodb.net/fitness-ai?retryWrites=true&w=majority

   IMPORTANT: Add "/fitness-ai" before the "?"
   
   Before: ...mongodb.net/?retryWrites...
   After:  ...mongodb.net/fitness-ai?retryWrites...

4. Save the file (Ctrl+S)

STEP 7: Backend Auto-Restarts
──────────────────────────────
✅ Nodemon will detect the .env change
✅ Backend will restart automatically
✅ You'll see: "✅ MongoDB Connected: cluster0.xxxxx.mongodb.net"

STEP 8: Try Registration Again
───────────────────────────────
1. Go back to: http://localhost:3000/register
2. Fill in the form:
   • Full Name: Your name
   • Email: Your email
   • Password: Your password
   • Confirm Password: Same password
3. Click: "Create Account"
4. ✅ SUCCESS! You'll be redirected to onboarding
