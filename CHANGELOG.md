# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.0] - 2024-01-01

### Added
- Initial release of FitAI platform
- User authentication system with JWT
- Health profile management
- Personalized diet plan generation
- Workout plan generation based on fitness goals
- Calorie prediction using ML algorithms
- BMR and TDEE calculations
- Progress tracking with weight logs
- Meal logging functionality
- Workout logging functionality
- AI health assistant chatbot
- Analytics dashboard with charts
- Light/Dark mode support
- Responsive design for all devices
- Docker containerization
- Comprehensive API documentation
- ML service with FastAPI
- MongoDB database integration
- Real-time data visualization with Recharts

### Features
- **Authentication**
  - User registration
  - User login
  - JWT token-based authentication
  - Password hashing with bcrypt
  - Protected routes

- **Health Profile**
  - Age, gender, height, weight tracking
  - Activity level selection
  - Fitness goal setting
  - Diet preference configuration
  - Workout experience level
  - Allergy and restriction management

- **Diet Planning**
  - Personalized meal plans
  - Calorie distribution across meals
  - Macro nutrient calculations
  - Hydration recommendations
  - Diet tips and suggestions

- **Workout Planning**
  - Weekly workout schedules
  - Exercise recommendations
  - Calorie burn estimates
  - Experience-based routines
  - Rest day planning

- **Progress Tracking**
  - Weight tracking over time
  - Meal logging with nutritional info
  - Workout logging
  - Water intake tracking
  - Progress analytics

- **AI Assistant**
  - Natural language chat interface
  - Fitness and nutrition guidance
  - Personalized insights
  - Weekly summaries
  - Health disclaimers

- **Analytics Dashboard**
  - Weight progress charts
  - Calorie intake visualization
  - Macro distribution pie charts
  - Goal progress tracking
  - Weekly statistics

### Technical
- React 18 with Vite
- Tailwind CSS for styling
- Framer Motion for animations
- Zustand for state management
- Node.js with Express backend
- MongoDB with Mongoose ODM
- Python FastAPI for ML service
- Docker and Docker Compose support
- NGINX reverse proxy configuration
