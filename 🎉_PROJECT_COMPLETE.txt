🚀 HOW TO START (3 SIMPLE WAYS):

METHOD 1: Automated Setup (Easiest)
────────────────────────────────────
Windows:
  1. Double-click: setup.bat
  2. Run: docker run -d -p 27017:27017 --name mongodb mongo:7.0
  3. Run: npm run dev
  4. Open: http://localhost:3000

Mac/Linux:
  1. Run: chmod +x setup.sh && ./setup.sh
  2. Run: docker run -d -p 27017:27017 --name mongodb mongo:7.0
  3. Run: npm run dev
  4. Open: http://localhost:3000

METHOD 2: Docker (One Command)
────────────────────────────────
  1. Run: docker-compose up --build
  2. Open: http://localhost:3000

METHOD 3: Manual Setup
────────────────────────────────
  1. Run: npm run install-all
  2. Run: cd ml-service && pip install -r requirements.txt
  3. Start MongoDB (Docker or local)
  4. Run: npm run dev
  5. Open: http://localhost:3000
